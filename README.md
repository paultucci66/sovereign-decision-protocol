# Sovereign Decision Protocol (SDP)

An open specification for representing, governing, preserving, and learning from consequential decisions — made by humans, AI systems, or the two acting together.

SDP gives organizations a common, auditable language for **who decided, what they were authorized to decide, which policies applied, what was actually done, and what was learned.** It records decisions; it does not execute them. That separation is what preserves accountability as autonomous agents take on more of the work.

## The Five Proofs

| Proof | Answers |
|-------|---------|
| Identity | WHO |
| Authority | MAY |
| Governance | SHOULD |
| Decision | DID |
| Memory | LEARNED |
